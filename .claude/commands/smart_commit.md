以下の手順でスマートコミットを実行してください：

1. `git status`と`git diff`を実行して変更内容を確認する

2. 変更内容を分析し、適切なコミットメッセージを生成する。形式：
   - feat: 新機能追加
   - fix: バグ修正
   - update: 既存機能の更新・改善
   - style: スタイル・デザインの変更
   - refactor: リファクタリング
   - docs: ドキュメントの変更
   - chore: その他の変更

3. 以下を順番に実行：
   - `git add .`
   - `git commit -m "生成したメッセージ"`
   - `git push`

4. 完了したら結果を報告する

注意：.envや機密情報が含まれていないか確認すること
