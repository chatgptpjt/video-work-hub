# 案件登録

新しい受注案件を登録します。

## 使い方
```
/案件登録
クライアント: ○○
案件内容: △△
報酬: ¥XX,XXX
納期: YYYY-MM-DD
```

## 参照ファイル
- テンプレート: 案件/案件テンプレート.md

## 手順

1. ユーザーから以下の情報を取得：
   - クライアント名
   - 案件内容
   - 報酬
   - 納期
   - その他要件

2. `案件/案件テンプレート.md` を元に新しいファイルを作成
   - ファイル名: `案件/YYYY-MM-クライアント名.md`

3. 作成したファイルの内容を表示

## 出力形式

```
✓ 案件を登録しました

ファイル: 案件/YYYY-MM-クライアント名.md
クライアント: ○○
報酬: ¥XX,XXX
納期: YYYY-MM-DD
```
