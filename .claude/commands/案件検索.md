# 案件検索・応募文作成

CrowdWorksで動画編集案件を検索し、応募文を作成します。

## 参照ファイル
以下のファイルを必ず読み込んでください：
- プロフィール: .claude/プロフィール/MET動画編集.md
- チーム情報: .claude/参考データ/チームアカウント.json

## 手順

1. **案件検索**
   - CrowdWorksで「動画編集」「ショート動画」「TikTok」「リール」などのキーワードで検索
   - URL例: https://crowdworks.jp/public/jobs?category=jobs&search_keywords=動画編集
   - WebFetchで検索結果を取得

2. **案件の絞り込み**
   以下の条件で案件を選定：
   - グルメ・飲食系優先
   - ショート動画（TikTok/リール/YouTube Shorts）
   - 継続案件歓迎
   - 単価が適正（1本3,000円〜）

3. **応募文の作成**
   選んだ案件に対して、プロフィール情報を元に応募文を作成：

   ```
   はじめまして！
   動画編集チーム【MET】と申します。
   ご依頼内容、拝見いたしました。

   [案件内容に合わせた具体的なアピール]

   ■ 私たちについて
   ・グルメ動画の実績50本超
   ・納品は2〜3日、週4〜5本まで対応可能
   ・チーム体制なので急ぎ案件も対応できます

   ■ ポートフォリオ
   現在運用中のアカウント：
   https://www.tiktok.com/@100mankai.food

   ぜひご検討いただけますと幸いです。
   よろしくお願いいたします！
   ```

4. **応募URLを提示**
   案件の応募ページURLを表示

## 出力形式

```
【案件タイトル】
URL: https://crowdworks.jp/...
報酬: ¥XX,XXX
内容: ...

【応募文】
（上記フォーマットで作成した応募文）

【応募方法】
上記URLにアクセスして応募文をコピー＆ペーストしてください。
```
