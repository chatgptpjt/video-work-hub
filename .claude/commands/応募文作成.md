# 応募文作成

指定されたCrowdWorks案件URLに対して、最適な応募文を作成します。

## 使い方
```
/応募文作成 [案件URL]
```

## 参照ファイル
以下のファイルを必ず読み込んでください：
- プロフィール: .claude/プロフィール/MET動画編集.md
- チーム情報: .claude/参考データ/チームアカウント.json

## 手順

1. **案件内容の取得**
   - 指定されたURLからWebFetchで案件詳細を取得
   - 以下を把握：
     - 案件タイトル
     - 報酬
     - 求められるスキル
     - 納期・本数
     - クライアントの求める人物像

2. **応募文のカスタマイズ**
   案件内容に合わせて応募文をカスタマイズ：
   - グルメ系 → グルメ動画実績をアピール
   - 継続案件 → チーム体制・週4-5本対応をアピール
   - 急ぎ案件 → 2-3日納品をアピール
   - 初心者歓迎 → 丁寧な対応をアピール

3. **応募文テンプレート**

   ```
   はじめまして！
   動画編集チーム【MET】と申します。
   ご依頼内容、拝見いたしました。

   [案件に合わせた具体的なアピール 2-3行]

   ■ 私たちについて
   ・グルメ動画の実績50本超
   ・納品は2〜3日、週4〜5本まで対応可能
   ・チーム体制なので急ぎ案件も対応できます
   ・修正は納得いただけるまで対応

   ■ ポートフォリオ
   現在運用中のアカウント：
   https://www.tiktok.com/@100mankai.food

   ご不明な点がございましたら、お気軽にお問い合わせください。
   ぜひご検討いただけますと幸いです。
   よろしくお願いいたします！
   ```

## 出力形式

案件情報と応募文をセットで出力してください。
応募文はそのままコピペできる形式で。
