# 作業ログ記録

今日の作業内容を記録します。

## 使い方
```
/作業ログ
やったこと: ○○
学んだこと: △△
明日やること: □□
```

## 参照ファイル
- テンプレート: 作業ログ/月次テンプレート.md

## 手順

1. 今月のログファイルを確認
   - ファイル名: `作業ログ/YYYY-MM.md`
   - なければテンプレートから作成

2. 今日の日付で作業ログを追記
   ```
   ### YYYY-MM-DD

   **今日やったこと**
   - （入力内容）

   **学んだこと・気づき**
   - （入力内容）

   **明日やること**
   - （入力内容）

   ---
   ```

3. 更新したファイルを保存

## 出力形式

```
✓ 作業ログを記録しました

ファイル: 作業ログ/YYYY-MM.md
日付: YYYY-MM-DD
```
