# 案件一覧

登録されている案件を一覧表示します。

## 手順

1. `案件/` ディレクトリ内のファイルを検索（テンプレート除く）

2. 各ファイルから以下を抽出：
   - クライアント名
   - 報酬
   - 納期
   - ステータス（進行中/完了）

3. ステータス別に一覧表示

## 出力形式

```
📋 案件一覧

【進行中】
1. クライアントA - ¥10,000 - 納期: 2026-01-20
2. クライアントB - ¥15,000 - 納期: 2026-01-25

【完了】
1. クライアントC - ¥8,000 - 完了: 2026-01-10

---
進行中: X件 / 完了: Y件 / 合計報酬: ¥ZZZ,ZZZ
```
