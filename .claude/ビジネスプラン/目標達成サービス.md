### 達成戦略の核心：保険型ビジネスモデル

従来のサブスクやスポット案件ではなく、**「保険型」の長期契約モデル**を採用する。

**なぜ保険型か？**
- サブスク → 途中解約されやすい（継続率：低）
- レンタル → 気軽にやめられる（継続率：中）
- **保険型** → 契約期間中は抜けにくい、お互い様の関係性（継続率：高）

> 「保険やったら途中で抜けたくないし、お互い様っていう関係性取れるんで、長期契約が取りやすい」— 足立さん

## 3. サービス設計

### サービス案1: AI導入＋運用保証パッケージ

**コンセプト**: システム導入だけでなく、運用を「保証」する保険をセットで提供

```
┌─────────────────────────────────────────────────┐
│        AI導入＋運用保証パッケージ               │
├─────────────────────────────────────────────────┤
│  初期費用: 50〜100万円（システム導入費）        │
│  月額費用: 5〜10万円（運用保証料）              │
│  契約期間: 12ヶ月〜（中途解約ペナルティあり）   │
└─────────────────────────────────────────────────┘
```

**提供内容**

| 項目 | 内容 |
|------|------|
| AIシステム導入 | 議事録AI、OCR見積書読み取り、業務自動化等 |
| 運用保証（12ヶ月） | 障害対応、軽微な改修、バージョンアップ対応 |
| 月次レポート | 利用状況・改善提案 |
| 優先サポート | 24時間以内レスポンス保証 |

**ターゲット顧客**
- 中小企業の経営者・管理職
- DXを推進したいがIT人材がいない企業
- 業務効率化に課題を感じている企業

**差別化ポイント**
- 導入後も「保証」があるので安心感がある
- 長期契約により関係性が深まり追加案件につながる

### サービス案2: グルメSNS運用代行＋成果保証

**コンセプト**: 動画編集チームの強みを活かした運用代行＋成果保証型

```
┌─────────────────────────────────────────────────┐
│       グルメSNS運用代行パッケージ               │
├─────────────────────────────────────────────────┤
│  月額費用: 15〜30万円                           │
│  契約期間: 6ヶ月〜                              │
│  成果保証: フォロワー増加目標 or 一部返金       │
└─────────────────────────────────────────────────┘
```

**提供内容**

| 項目 | ライトプラン（15万円） | スタンダードプラン（30万円） |
|------|------------------------|------------------------------|
| 動画本数 | 週3本（月12本） | 週5本（月20本） |
| 企画・撮影 | 素材提供のみ | 企画〜撮影まで対応 |
| 編集 | 基本編集 | 高品質編集＋効果音・テロップ |
| レポート | 月1回 | 週1回＋改善提案 |
| 成果保証 | なし | フォロワー増加目標設定 |

**ターゲット顧客**
- 飲食店・グルメ系インフルエンサー
- 食品メーカー・食品EC
- 地方自治体（観光・特産品PR）

**成果保証の設計**
- 6ヶ月でフォロワー○○人増加を目標設定
- 未達成時は翌月の料金を30%割引
- 達成時は継続契約へのインセンティブ

### サービス案3: ホールディングス型運営

**コンセプト**: 丸岡さんは統括・営業に専念し、専門チームを束ねる

```
┌─────────────────────────────────────────────────┐
│            丸岡（統括・営業）                   │
│         ┌──────────────────────┐                │
│         │  売上の20-30%をマージン  │                │
│         └──────────────────────┘                │
├─────────────────────────────────────────────────┤
│  ┌───────────┐  ┌───────────┐  ┌───────────┐  │
│  │ 動画編集  │  │  AI開発   │  │ コンサル  │  │
│  │ チーム    │  │  チーム   │  │  チーム   │  │
│  │（業務委託）│  │（業務委託）│  │（業務委託）│  │
│  └───────────┘  └───────────┘  └───────────┘  │
└─────────────────────────────────────────────────┘
```

**ポイント**

| 項目 | 内容 |
|------|------|
| 丸岡さんの役割 | 営業・クライアント窓口・品質管理 |
| メンバーの立場 | 「独立」ではなく「業務委託」として参画 |
| 収益配分 | 案件売上の70-80%をメンバーへ、20-30%を丸岡さんへ |
| スケール性 | チーム増設により売上拡大が可能 |

**メリット**
- 丸岡さん自身が作業に追われない
- メンバーは安定した仕事を得られる
- チーム拡大により収益が指数関数的に増加

## 5. 実行計画

   - 実績作りが必要

3. **第三優先**: ホールディングス型への移行

## 6. リスクと対策

### リスク1: チームメンバー離脱

| リスク | 影響度 | 対策 |
|--------|--------|------|
| メンバーが独立してしまう | 高 | 業務委託契約で明確化、継続的な仕事提供 |
| 品質にばらつきが出る | 中 | マニュアル整備、品質チェック体制 |

### リスク2: 顧客獲得の難しさ

| リスク | 影響度 | 対策 |
|--------|--------|------|
| 高単価サービスが売れない | 高 | まずは低単価で実績作り、段階的に単価アップ |
| 競合との差別化ができない | 中 | 「保証」を前面に出した安心感の訴求 |

### リスク3: 成果保証の未達

| リスク | 影響度 | 対策 |
|--------|--------|------|
| フォロワー増加目標未達 | 中 | 現実的な目標設定、達成可能なKPI設計 |
| 返金により利益が減る | 中 | 返金上限を設定（月額の30%まで等） |
